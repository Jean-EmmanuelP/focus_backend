# La Problématique

## Le constat brutal

```
92% des gens n'atteignent pas leurs objectifs annuels.
80% abandonnent leurs résolutions avant février.
La volonté seule ne suffit pas.
```

## Pourquoi les solutions actuelles échouent

### 1. Les apps de productivité sont PASSIVES

```
User → Doit penser à ouvrir l'app
     → Doit naviguer
     → Doit logger manuellement
     → Doit s'auto-motiver

Résultat: App oubliée dans un dossier après 2 semaines
```

**Taux de rétention moyen des apps productivité:**
- J1: 25%
- J7: 10%
- J30: 3%

### 2. Pas de responsabilisation

```
Seul face à son écran:
- Personne ne sait si tu as fait ta session
- Pas de conséquence à l'abandon
- Facile de se mentir à soi-même
```

**Ce qui marche dans le monde réel:**
- Coach sportif: quelqu'un t'attend
- Groupe de travail: pression sociale positive
- Deadline externe: conséquence réelle

### 3. Friction cognitive élevée

```
Pour logger une session de focus:

App classique:           WhatsApp:
1. Déverrouiller         1. Envoyer "focus 30min"
2. Trouver l'app
3. L'ouvrir              Terminé.
4. Naviguer au bon écran
5. Cliquer sur +
6. Remplir le formulaire
7. Valider

~30 secondes + friction   ~3 secondes
```

### 4. Motivation qui s'essouffle

```
Semaine 1: "Je vais tout déchirer!"
Semaine 2: "C'est dur mais je tiens"
Semaine 3: "J'ai oublié hier..."
Semaine 4: "Bon, j'arrête"
```

**Pourquoi?**
- Pas de feedback immédiat
- Pas de célébration des petites victoires
- Pas d'adaptation au contexte (fatigue, stress, vie)

---

## La vraie question

> "Comment créer un système qui vient vers l'utilisateur,
> le comprend, et l'accompagne sans friction?"

---

## L'insight clé

```
Les gens n'échouent pas par manque de volonté.
Ils échouent par manque de SYSTÈME.

Un bon système:
✅ Est proactif (vient vers toi)
✅ Est adaptatif (comprend ton contexte)
✅ Est présent (toujours accessible)
✅ Est bienveillant mais exigeant
✅ Célèbre ET challenge
```

---

## Pourquoi WhatsApp?

### Statistiques d'usage

| Métrique | WhatsApp | Apps productivité |
|----------|----------|-------------------|
| Ouvertures/jour | 23x | 0.5x |
| Temps de réponse | <2min | Jamais |
| Taux d'ouverture notif | 98% | 15% |
| Désinstallation J30 | <1% | 70% |

### Comportement naturel

```
WhatsApp = Déjà dans la routine quotidienne
         = Notifications activées
         = Réflexe de répondre
         = Conversation naturelle

App dédiée = Effort conscient pour ouvrir
           = Notifications ignorées/désactivées
           = Pas de réflexe
           = Interface à apprendre
```

                      │  └─────────────┘

                      │  │ Apps passives│ ← Pas efficace
                      │  └─────────────┘

**Focus se positionne dans le gap:**
- Plus efficace que les apps (proactif + IA)
- Moins cher qu'un coach (14.99€ vs 200€+)
- Disponible 24/7 (vs 1h/semaine)
