# Stratégie Go-to-Market

## Positionnement

```
"Ton coach de vie personnel dans WhatsApp.
 Pas une app de plus. Un ami qui te pousse."
```

### Message clé

```
Tu n'as pas besoin d'une nouvelle app.
Tu as besoin de quelqu'un qui te comprend,
te challenge, et célèbre tes victoires.

Kai est là. Dans WhatsApp. 24/7.
```

---

## Cible principale

### Persona primaire: "L'Ambitieux Débordé"

```
Démographie:
- 25-40 ans
- Actif professionnel (startup, freelance, cadre)
- Urbain (grandes villes)
- Revenus: 35-80k€/an
- Mobile-first

Psychographie:
- A des objectifs personnels (fitness, side-project, lecture)
- Manque de temps/discipline pour les tenir
- A essayé des apps de productivité (abandonné)
- Utilise WhatsApp quotidiennement (pro + perso)
- Prêt à payer pour des solutions qui marchent
- Frustré par son manque de constance

Pain points:
- "Je commence motivé, j'abandonne après 2 semaines"
- "Je n'ai pas le temps de gérer une app de plus"
- "Je me mens à moi-même sur mes progrès"
- "J'ai besoin de quelqu'un pour me tenir accountable"
```

### Persona secondaire: "L'Étudiant Organisé"

```
Démographie:
- 18-25 ans
- Étudiant supérieur
- Budget limité mais priorise le développement personnel

Psychographie:
- Prépare sa carrière
- Consomme du contenu productivité (YouTube, podcasts)
- Cherche des outils pour se démarquer
- Influencé par les pairs et les créateurs
```
